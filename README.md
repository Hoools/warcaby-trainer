# Warcaby 100-polowe Trainer

Narzędzie do nauki gry w międzynarodowe warcaby (10x10).

## Funkcje
- Interaktywna plansza
- Walidacja ruchów
- Termometr prawidłowości ruchu
- Pokaz najlepszych 3 ruchów z rzędu
- Później: tryb AI

## Uruchomienie
1. Otwórz `index.html` w przeglądarce.

## Setup GitHub
1. `cd warcaby-trainer`
2. `git init`
3. `git add .`
4. `git commit -m "Initial commit"`
5. Utwórz repo na GitHub (np. nazwij "warcaby-trainer")
6. `git remote add origin https://github.com/TWOJ_USERNAME/warcaby-trainer.git`
7. `git branch -M main`
8. `git push -u origin main`
